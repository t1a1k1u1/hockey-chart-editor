# Assets

**Art direction:** Dark-theme professional DAW/chart editor UI. Dark charcoal backgrounds (#1E1E2E), track rows with subtle color-coded backgrounds (orange tones for TOP rows, dark blue for NORMAL, darker blue for VERTICAL rows). Note blocks rendered as solid colored rectangles with rounded caps — blue (#4D66FF) for normal, green (#1AE64D) for top, purple (#8855FF) for vertical. Red playhead vertical line. Clean monospace-like labels. Toolbar with flat buttons.
